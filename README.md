# C++ Posion (Pocket Conversion)

Tugas besar algoritma pemrograman

## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/Apndw/cpp-tubes-alpro.git
~~~

Go to the project directory  

~~~bash  
  cd cpp-tubes-alpro
~~~

Build the cpp file

~~~bash  
  g++ -std=c++11 ./main.cpp  -o ./output/main
~~~

Start the project

~~~bash  
  ./output/main
~~~

Or you can use bash script

~~~bash
  ./run.sh
~~~
